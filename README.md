## ✨ Tính Năng Nổi Bật (Features)
- **Giao diện chuẩn UI/UX:** Cấu trúc bằng HTML5 chuẩn semantic và dàn trang bằng CSS Flexbox hiện đại.
- **Dark/Light Mode Toggle:** Tính năng chuyển đổi giao diện Sáng/Tối mượt mà thao tác trực tiếp vào DOM.
- **Render Dữ Liệu Động (Dynamic Rendering):** Sử dụng JavaScript (ES6) để tự động xuất danh sách kỹ năng từ dữ liệu Array of Objects.

## 🛠️ Công Nghệ Sử Dụng (Tech Stack)
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6).
- **Công cụ:** VS Code, Git, GitHub.
- **Deploy:** GitHub Pages.
